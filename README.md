# Privy Expo Starter

This demonstrates a minimal working installation of the privy sdk in a fresh
expo app.

## Setup

1. Install dependencies

   ```sh
   npm i
   ```

1. Add your Privy app ID in `app.json`

   ```json
   ...
    "extra": {
      "privyAppId": "<your-app-id>"
    }
   ...
   ```

## Run the app

```sh
# expo go
npm start

# ios
npm run ios

# android
npm run android
```
